# Object-Detection
Allows user detect objects via webcam.

> Tests internet.

> Tests video.

> Tests audio.

## Built with NextJS, Tailwind CSS, Hooks and Tensorflow

## Install

```sh
cd .\object-detection\
npm install
```

## Usage

```sh
cd .\object-detection\
npm run dev
```

Spins up local server on port 3000
